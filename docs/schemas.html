--- layout: default title: Schema Documentation root: . description: 'Schemas are used to add structure to data. This schema is based off the Material Schema and represents a guide for how to structure data for materialhub and how to interpret data seen on materialhub. This helps ensure that the meaning of the data is preserved, ensures the data remains FAIR, and assists collaboration. The schemas only serve as a guide and materialhub is flexible enough to accept any data. However, to add meaning to new materialhub properties requires creating DefinedTerms, a slightly more complicated process. Therefore, the schema helps with rapid data ingestion.' ---
